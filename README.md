# Professional Landing Page Project

This repository contains a professional, responsive landing page built with modern web standards.

Project structure
- `index.html` — main page
- `calculator.html` — calculator page
- `projects.html` — projects/list page
- `script.js` / `style.css` — shared JS and CSS
- `assets/` — images and other assets
- `en/` — English-localized copies of the same site

Tech stack
- HTML5 — semantic markup and accessibility
- CSS3 — responsive layout (Flexbox / Grid)
- JavaScript (ES6) — interactive behavior and DOM handling
- Static — no build tools required (plain HTML/CSS/JS)

How to run (quick):
1. Open `index.html` in your browser, or run a local static server.

Run a simple Python server (from project root):
```bash
python -m http.server 8000
```
Then open http://localhost:8000 in your browser.

Or with Node (if you have http-server installed):
```bash
npx http-server -p 8000
```

Notes
- This is a static site — no build step required.
- For changes to JS/CSS, refresh the browser to see updates.
